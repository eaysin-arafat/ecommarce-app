# Ecommerce App

This is a simple e-commerce application built with React, Redux Toolkit, and JSON Server.

## Prerequisites

Before running the project, make sure you have [Node.js](https://nodejs.org/) installed on your machine.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/ecommerce-app.git

    Navigate to the project directory:

    bash

cd ecommerce-app

Install dependencies:

bash

    yarn install

Usage

To start the development server, run:

bash

yarn dev

This command will start the Vite development server.

To start the JSON Server for database, run:

bash

yarn db

This command will start the JSON Server with authentication.
Available Scripts

    yarn dev: Start the Vite development server.
    yarn build: Build the project for production.
    yarn lint: Run ESLint to lint your JavaScript code.
    yarn preview: Preview the production build locally.
    yarn db: Start the JSON Server for database with authentication.

Dependencies

    @reduxjs/toolkit: A package to manage Redux state.
    axios: Promise based HTTP client for the browser and node.js.
    dotenv: Loads environment variables from .env file.
    json-server: A full fake REST API with zero coding.
    json-server-auth: Authentication module for JSON Server.
    react: A JavaScript library for building user interfaces.
    react-dom: React package for working with the DOM.
    react-icons: Icons for React projects.
    react-redux: Official React bindings for Redux.
    react-router-dom: Declarative routing for React.
